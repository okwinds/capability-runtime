# Examples Guide

Start here if you want runnable entrypoints:

- `examples/01_quickstart/`: smallest runtime loop
- `examples/02_workflow/`: sequential, loop, and conditional orchestration
- `examples/03_bridge_e2e/`: real backend bridge path
- `examples/04_triggerflow_orchestration/`: host-level orchestration around `Runtime.run()`
- `examples/05_workflow_skills_first/`: workflow composition with skills-first agents
- `examples/apps/`: terminal, HTTP/SSE, and UI-style examples

For coding-agent-focused examples, see [docs_for_coding_agent/examples/README.md](../docs_for_coding_agent/examples/README.md).
